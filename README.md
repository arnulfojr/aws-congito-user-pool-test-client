# cognito-test

It's based on Vue and AWS Cognito + CDK

Make sure the env variables set in the setup.sh file are correct for you.
Make sure you have the awscli set up, as well!

To set up:

```
./setup.sh
# does npm i
# then cdk deploy
# then starts vue as a local devserver
```
